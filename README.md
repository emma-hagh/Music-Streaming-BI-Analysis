# 🎵 SoundWave — Music Streaming Business Intelligence Dashboard

## 📌 Project Overview

SoundWave is a Business Intelligence and Data Visualization project developed using Power BI.

The project analyzes the performance of a fictional music streaming platform inspired by services such as Spotify and Deezer. The objective is to transform raw streaming, artist, subscriber, and royalty data into actionable business insights that support strategic decision-making.

The dashboard combines data modeling, DAX calculations, KPI analysis, and interactive visualizations to monitor platform growth, artist performance, monetization efficiency, and strategic opportunities.

---

# 🎯 Business Objectives

* Analyze streaming activity and user engagement
* Evaluate artist and genre performance
* Monitor monetization efficiency
* Track royalties and revenue generation
* Identify strategic growth opportunities
* Support executive decision-making through interactive dashboards

# 🧠 Data Modeling

The project uses a Star Schema architecture composed of:

* Central fact table (`daily_streams`)
* Artist dimension
* Track dimension
* Album dimension
* Subscriber dimension
* Dedicated Date Table for time intelligence

This structure enables:

* Efficient filtering
* Scalable KPI calculations
* Reliable time-series analysis
* Optimized Power BI performance
